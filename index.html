NoAI: Browser-Only Document Redaction for AI Tools

NoAI helps people prepare documents before using external AI tools. It reads supported files in the browser, applies deterministic redaction rules locally, and exports redacted Markdown for review and sharing.

What NoAI Does

  • Reads Markdown, plain text, Word documents, and text-based PDFs in the browser.
  • Finds likely sensitive details such as names, emails, phone numbers, dates, amounts, addresses, case references, organizations, locations, and custom terms.
  • Exports combined Markdown so users can paste reviewed, redacted text into tools such as ChatGPT, Claude, Gemini, or other AI assistants.

Trust Boundaries

NoAI's redaction path does not upload document contents, does not call an AI model, and does not use analytics or telemetry that can inspect document text. Users should still review the output before sharing it, especially for legal, financial, medical, or other high-risk documents.
